# Simplifying SAS Viya: Explaining the Compute and CAS Servers, Caslibs, and In-Memory Data
Simplifying SAS®  Viya® : Explaining the Compute and CAS Servers, Caslibs, and In-Memory Data

# Abstract 
SAS Viya is a modern, cloud-enabled analytics platform built for high-speed, scalable processing- but what does that mean in practice for SAS programmers? This demonstration demystifies SAS Viya by explaining its core architecture, introducing caslibs, and showing how they compare to traditional SAS libraries. You’ll also see how to load data into memory and explore key considerations when working with the CAS server. Whether you’re new to SAS Viya or transitioning from SAS9, this session provides a practical foundation for working effectively in SAS Viya.
